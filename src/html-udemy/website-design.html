Website Design with Google Sites

ICT Skills for Professional Development

Explore the course outline prepared to give you a strong foundation in mastery

Course Description
  • Learn how to create professional looking websites by using the super friendly and easy to use Google Sites, a free website builder from Google.
  • It does not require you to do program coding and you can choose a theme and customize it any way you want for a professional-looking site that looks and functions great across all desktop and mobile screens.
  • It has site-builder features that make it easy and intuitive to design any website the way you want. There are plentiful add-ons such as text, photos, videos, calendars, maps, spreadsheets, presentations and more to make your site functional and rich in content.
  • You will be surprised to learn what you can do by just spending a few hours of your time learning Easy Website Design with Google Sites.
What will students learn in the course?
  • Setup a website using Google Sites
  • Organise the website navigation structure
  • Develop and design webpage content
  • Use Google Sites Add-on components
  • Create online forms to collect user feedback
Are there any course requirements or prerequisites?
  • Interested to explore new ICT skills
Who can join the course?
  • Open to everyone who is interested to learn how to develop a website.
